HERO VIDEO (optional)
=====================

Add your hero background video here:
  hero.mp4

Tips:
- Keep file size under 8 MB for fast loading
- Use landscape 1920×1080, 10–20 seconds, looping
- Muted ocean / ship / sunset footage works best
- Also add hero.jpg in public/images/ as poster/fallback (required for mobile & SEO)

If no video is present, the site uses hero.jpg automatically.
